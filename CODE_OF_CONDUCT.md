# Code of Conduct

This project is committed to providing a respectful, inclusive, and harassment-free environment.

Participants are expected to:

- Be kind and constructive
- Respect different community contexts
- Avoid personal attacks
- Credit ideas and contributions
- Handle disagreement with care

Unacceptable behaviour includes harassment, intimidation, discrimination, sustained disruption, or sharing private information without consent.

Reports and enforcement process to be added by project maintainers.
